About Grocery Management System

Welcome to our Grocery Management System! This system is designed to streamline the operations of your grocery store, making it easier to manage products, orders, and customer interactions.

Features

  • Product Management: Easily add, edit, and delete products and categories.
  • Order Management: Track orders, manage statuses, and handle customer requests.
  • User Authentication: Secure login system with roles for admin and regular users.
  • Cart and Checkout: Smooth shopping cart experience with checkout functionality.
  • Analytics and Reporting: Generate reports on sales, inventory, and customer trends.

Technologies Used

  • Python Flask framework for backend development.
  • SQLAlchemy for database management and ORM.
  • HTML, CSS, and JavaScript for frontend development.
  • GitHub for version control and collaboration.
